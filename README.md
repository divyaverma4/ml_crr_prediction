# AMEX Shadowing Program Project

## Overview
Using **Machine Learning (ML)** to determine **Customer Risk Rating (CRR)** in real-world scenarios, accounting for **outdated or missing data**. The goal is to predict CRR in a way that mirrors **corporate processes**.

## Aim
- Predict CRR realistically for large-scale applications.
- Handle incomplete or outdated data effectively.

## Tech Stack
- Python
- Pandas, NumPy, scikit-learn
- Flask, SQLite
- HTML

## Quick Start
1. Clone the repo:  
   ```bash
   git clone https://github.com/yourusername/ml_crr_prediction.git
   cd credit_risk
   ```
2. Setup virtual environment:  
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   source .venv/bin/activate  # macOS/Linux
   ```
3. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
4. Run the project:  
   ```bash
   python main.py
   ```
   Or start the Flask app:  
   ```bash
   flask run
   ```
